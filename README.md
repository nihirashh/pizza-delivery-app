# 🍕 Pizza Delivery Application

A full-stack Pizza Delivery Application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).  
The project includes secure user authentication, pizza customization, Razorpay payment integration, inventory management, and an admin dashboard for order tracking and stock management.

---

## 🚀 Features

- User Registration & Login Authentication
- JWT-based Authorization
- Forgot Password & Email Notification System
- Interactive Pizza Customization
- Cart & Order Management
- Razorpay Payment Gateway Integration (Test Mode)
- Admin Dashboard
- Inventory & Stock Management
- Order Status Tracking
- Low Stock Email Alerts

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
